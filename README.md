# Asset Predictor

Aplicación web desarrollada con **ASP.NET Core MVC (.NET 9)** para la predicción de tendencias de activos financieros (acciones y criptomonedas) utilizando distintos modelos de análisis técnico y estadístico.

---

## 🚀 Funcionalidades principales

### 📊 Predicción de tendencias financieras
- Predicción de comportamiento de activos financieros
- Procesamiento de series históricas de precios
- Análisis de tendencias alcistas y bajistas

### 🧠 Modos de predicción
- Media Móvil Simple (SMA Crossover)
- Regresión Lineal
- Momentum (ROC)

### 📝 Gestión de datos históricos
- Registro de 20 días históricos del activo
- Validación de series de datos
- Entrada manual mediante formulario o textarea

### ⚙️ Configuración dinámica
- Selección de modo de predicción
- Persistencia del modo seleccionado en memoria
- Cálculo dinámico según el algoritmo seleccionado

---

## 🧠 Aspectos destacados del proyecto

- Implementación de algoritmos financieros y matemáticos
- Arquitectura MVC con separación de responsabilidades
- Validaciones mediante ViewModels
- Uso de DTOs para transferencia de datos
- Procesamiento y análisis de series temporales
- Interfaz web responsive y estructurada

---

## 🧩 Tecnologías usadas

- ⚙️ ASP.NET Core MVC (.NET 9)
- 📊 Algoritmos de análisis financiero
